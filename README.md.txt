----packages to be installed----
1)Image processing
pip install opencv-python 

2) Mediapipe module  
pip install mediapipe 

3) To control volume(Windows) 
pip install pycaw

Note: make sure the above modules are installed before running the project to avoid errors.

----steps to run the project----
1) Open the project using any of the IDE(I have used PyCharm)
2) Navigate to the HandTracker.py file to run the file
3) The expected result of step 2 is to mark landmarks with connections on the user palm
4) After successful output from step 2, navigate to volcon.py and run it. 
5) Now the user will be able to alter the system audio by distancing using thumb and index fingers
